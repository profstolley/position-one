Understanding the basics of widths in CSS

Default Widths

By default, all block-level elements are as wide as their containing element that has a specified width. If no containing element has a specified width, than the element is as wide as the browser window. See this in action by putting the following CSS into the Web Developer Add-on's CSS editor: 

div.containing-element-example {
  background-color: red; 
}

Nifty. Now, change the containing element on the page (div#page) to be 800 pixels wide: 

div#page {
  width: 800px;
}

This box should now appear 800 pixels wide, because it is filling the entire 800 pixels of the containing div#page element.

Specified Widths

Widths can be specified in two basic ways. The first is to have what amounts to an automatic width, by specifying left and/or right margins. Try sizing this area of the page by putting the following CSS into the Web Developer Add-on's CSS editor: 

div.specified-width-example {
  background-color: green;
  margin-right: 200px;
  margin-left: 300px;
}

Alternatively, you can specify a width using the width property. Remove the CSS you wrote above, and try setting this area of the page to be 300 pixels wide: 

div.specified-width-example {
  background-color: green;
  width: 300px;
}

Borders, Padding, and Widths

So far, so good. However, widths get a little more complicated when you start adding in borders and padding. Borders and padding are added to the widths of elements that have a specified width. Try, for example, 

div.border-padding-example {
  background-color: blue;
  width: 300px;
}

Then, to that add padding: 50px;. Then add border: 50px solid black;. You'll end up with this box being 500 pixels wide: 300px width + 100px padding (50px/each right and left) + 100px border (50px/each right and left) = 500px.

Absolute Positioning and Widths

Absolutely positioned elements shrink to the width of content.

See for yourself:

div.absolute-width-example {
  background-color: yellow;
  position: absolute;
}

Not the right size? Specify width:

div.absolute-width-example {
  background-color: yellow;
  position: absolute;
  width: 500px;
}
